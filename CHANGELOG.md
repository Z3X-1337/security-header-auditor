# Changelog

All notable changes to this project will be documented in this file.

The project follows Semantic Versioning.

## [0.1.0] - 2026-07-12

### Added
- Selected HTTP security-header auditing.
- Redirect-chain and final-response metadata.
- HEAD-to-GET fallback for HTTP 405 and 501.
- Baseline CSP directive analysis.
- Schema-versioned JSON reports.
- Stable CI exit codes and `--fail-under`.
- Unit, CLI, and local integration tests across Python 3.10, 3.11, and 3.12.
- Installable command-line entry point.
