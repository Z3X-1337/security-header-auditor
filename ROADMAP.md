# Roadmap

## v0.2.0
- Expand CSP directive parsing and reporting.
- Add cookie-security observations.
- Add TLS metadata collection.
- Add Markdown output.

## v0.3.0
- Add configuration profiles for CI policies.
- Add route-list input for authorized applications.
- Add SARIF export.
- Add comparative reports between deployments.

## Non-goals
- Broad internet scanning.
- Claiming vulnerability or exploitability from header presence alone.
- Replacing application security testing.
